# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**DO NOT open a public GitHub issue.**

Instead, email: security@openfihris.dev

We will acknowledge your report within 48 hours and work with you to understand and fix the issue before any public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |
